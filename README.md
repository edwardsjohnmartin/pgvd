To build with CMake
------------------------------------------------------------
* On Linux/Mac
  1. mkdir release
  2. cd release
  3. cmake ..
  4. make

* On Windows
	1. Install Visual Studio 2017
	2. Download GLM, GLEW, and GLFW.
	3. Create the following environment variables:
		* GLEW_LOCATION
		* GLFW_LOCATION
		* GLM_ROOT_DIR
	4. Use the "Team Explorer" to clone this repository
